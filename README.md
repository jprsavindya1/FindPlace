# Find Place - Booking Platform

A premium booking platform for hotels, villas, restaurants, and more.

## Project Structure

This project is divided into two main parts:

- **`frontend/`**: React-based user interface using Vite, Framer Motion, and Lucide React.
- **`backend/`**: Express.js server with MySQL database integration.

## Getting Started

### Backend
1. Navigate to the `backend` directory: `cd backend`
2. Install dependencies: `npm install`
3. Configure environment variables in `.env`
4. Start the server: `node server.js`

### Frontend
1. Navigate to the `frontend` directory: `cd frontend`
2. Install dependencies: `npm install`
3. Start the development server: `npm start`

## Technologies Used
- **Frontend**: React, Framer Motion, Lucide React, Axios, CSS Modules (Luna theme)
- **Backend**: Node.js, Express, MySQL, Multer, JWT
- **Design**: Premium "Luna" theme with glassmorphism and modern aesthetics.
